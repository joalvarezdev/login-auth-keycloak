# Login Auth Keycloak

## Descripción

Proyecto full stack con Angular 17 en el frontend y Spring Boot con Java 17 en el backend, la seguridad de la aplicación en manejada con Keycloak.

## 🛠️ Stack

- Angular 17
- Docker
- Spring Boot | Java 17
- PostgreSQL
- Liquibase
- Keycloak
